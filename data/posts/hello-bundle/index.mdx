This entire post lives inside a single folder, `data/posts/hello-bundle/`:

```
data/posts/hello-bundle/
  index.mdx          # this file (English original)
  index.ar.mdx       # Arabic translation
  images/cover.png   # → /static/images/posts/hello-bundle/cover.png
  demos/counter.html # → /demos/posts/hello-bundle/counter.html
```

To add a new post you drop a folder like this in — nothing else in the app needs
to change. `scripts/sync-content.mjs` mirrors `images/` and `demos/` into `public/`
before every dev/build, and the section is set in frontmatter (`section: 'free-writing'`).

## Portable demo

Interactive demos ship as self-contained `.html` files in the bundle's `demos/`
folder and embed via the `<Demo />` component — no app rebuild, no shared component:

<Demo src="/demos/posts/hello-bundle/counter.html" title="Counter demo" height={220} />

That's the whole convention. See `data/posts/README.md` for the full reference.
